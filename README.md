Encryption and Decryption Tool (Tkinter)
  A Python-based GUI application to securely encrypt and decrypt text or files using a user-defined key. Built with Tkinter for a simple and interactive experience.

Features
- Key-based encryption & decryption
- ASCII-safe text encryption using Base64 (no Unicode issues)
- Supports encryption of all file types (.enc format)
- Save and load text files
- Automatically handles both text and file operations
- Clean and user-friendly GUI

Tech Stack
- Python
- Tkinter
- Base64

How to Run
python main.py

Note
- Use the same key for encryption and decryption; otherwise, output will be unreadable.
